# Guide

Welcome to the Starfish guide. This section covers everything you need to get started with the Starfish protocol.

## What is Starfish?

Starfish is a transport-neutral realtime protocol designed for creative coding. It provides sessions, presence tracking, pub/sub messaging, shared data, and peer-to-peer connectivity for networked performance, multiplayer sketches, installations, and distributed browser-based artworks.
