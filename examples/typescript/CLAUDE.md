# Starfish TypeScript Examples

Example applications demonstrating Starfish TypeScript SDK usage.

## Verification

Before considering any changes complete, run the following checks from this directory (`examples/typescript/`):

```bash
# Compile (must pass with no errors)
npm run build

# Lint
npm run lint

# Format check
npm run format:check
```

All commands must exit with code 0. Do not skip any of them.

If formatting fails, run `npm run format` and include the formatting fixes in your changes.
