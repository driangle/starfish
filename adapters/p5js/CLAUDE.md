# Starfish p5.js Adapter

Adapter that integrates the Starfish TypeScript SDK with p5.js sketches.

## Verification

Before considering any changes complete, run the following checks from this directory (`adapters/p5js/`):

```bash
# Type-check (must pass with no errors)
npm run check

# Lint
npm run lint

# Format check
npm run format:check

# Unit tests
npm run test
```

All four commands must exit with code 0. Do not skip any of them.

If formatting fails, run `npm run format` and include the formatting fixes in your changes.
